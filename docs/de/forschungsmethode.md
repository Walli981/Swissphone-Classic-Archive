# Forschungsmethode

Jede Erkenntnis wird einer von drei Kategorien zugeordnet:

## Beobachtung

Direkt gemessen oder auf einem Foto eindeutig erkennbar.

## Interpretation

Eine durch mehrere Beobachtungen gestützte technische Einordnung.

## Arbeitshypothese

Eine plausible, aber noch nicht bestätigte Erklärung.

Jeder Versuch erhält eine feste Kennung wie `EXP-0001`. Messparameter, Anschlussbelegung, Rohdaten und Schlussfolgerung werden gemeinsam dokumentiert.
