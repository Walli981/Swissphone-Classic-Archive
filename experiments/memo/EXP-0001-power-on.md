# EXP-0001 — Power-on, Pin 2 and Pin 3

## Deutsch

**Anschlüsse:** CH0 = Pin 2, CH1 = Pin 3  
**Ergebnis:** Während des Einschaltens traten kurze Übergänge auf Pin 2 auf. Pin 3 wechselte später dauerhaft auf High. Es wurde kein eindeutig gültiges UART-Telegramm bestätigt.

## English

**Connections:** CH0 = pin 2, CH1 = pin 3  
**Result:** Short transitions occurred on pin 2 during power-up. Pin 3 later changed permanently to high. No clearly valid UART frame was confirmed.
