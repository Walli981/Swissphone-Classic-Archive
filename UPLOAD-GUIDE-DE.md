# Upload-Anleitung für GitHub

1. ZIP-Datei lokal entpacken.
2. Den Ordner öffnen und **nur dessen Inhalt** auswählen.
3. Das GitHub-Repository öffnen.
4. `Add file` → `Upload files` wählen.
5. Alle entpackten Dateien und Ordner in das Uploadfeld ziehen.
6. Prüfen, dass keine zusätzliche Ordnerebene `Swissphone-Classic-Archive-update-v0.3.1/` im Repository entsteht.
7. Commit-Nachricht verwenden:

   `Add bilingual v0.3.1 documentation update`

8. Änderungen committen.

Vorhandene Dateien wie `README.md`, `CONTRIBUTING.md` und `ROADMAP.md` werden aktualisiert. Rohdaten und Fotos werden in neue Unterordner eingefügt.
