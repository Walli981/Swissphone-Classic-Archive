# Mitwirken / Contributing

## Deutsch

Beiträge sind willkommen, besonders:

- eigene Gerätefotos
- reproduzierbare Messungen
- Bauteilidentifikationen
- Schaltpläne
- Korrekturen
- historische Hinweise mit nachvollziehbarer Quelle

Bitte trenne deutlich zwischen:

1. **Beobachtung**
2. **Interpretation**
3. **Arbeitshypothese**

Rohdaten sollten unverändert hochgeladen werden. Ergänze dazu einen Messbericht anhand der Vorlage unter `templates/experiment-template.md`.

## English

Contributions are welcome, especially:

- original device photographs
- reproducible measurements
- component identifications
- schematics
- corrections
- historical references with traceable sources

Please clearly separate:

1. **Observation**
2. **Interpretation**
3. **Working hypothesis**

Raw data should remain unchanged. Add an experiment report based on `templates/experiment-template.md`.
