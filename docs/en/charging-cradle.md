# Swissphone Memo charging cradle

## Visible features

The PCB includes a BNC socket and several logic ICs. Earlier descriptions of the connector as RJ45 or USB were incorrect and have been corrected.

## Current status

The function of the BNC socket has not yet been verified. Possible uses such as an antenna connector or RF service access remain hypotheses until the center conductor has been electrically traced.

## Planned measurements

- Verify BNC shield connection to ground
- Trace the BNC center conductor to device contacts and IC pins
- Number and map all spring contacts
- Reconstruct a schematic from traces and component values
