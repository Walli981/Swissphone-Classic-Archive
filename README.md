# 🇩🇪 Deutsch

## Willkommen

Das **Swissphone Classic Archive** ist ein freies Open-Source-Projekt zur Dokumentation historischer Swissphone-Funkmeldeempfänger.

Ziel des Projektes ist die langfristige Sicherung von Wissen, Software, Dokumentationen, Messungen und technischen Erkenntnissen rund um klassische Swissphone-Geräte.

Viele historische Swissphone-Geräte, Programmiertools und technische Unterlagen werden zunehmend schwer zugänglich. Dieses Projekt möchte dieses Wissen bewahren, bevor es verloren geht.

### Projektziele

- Dokumentation historischer Swissphone-Geräte
- Archivierung von Software und Handbüchern
- Reverse Engineering der Hardware
- Dokumentation reproduzierbarer Messungen
- Entwicklung freier Werkzeuge
- Internationale Zusammenarbeit

> **Dieses Projekt bewahrt Technikgeschichte. Unser Ziel ist es, historische Swissphone-Geräte durch reproduzierbare Messungen, sorgfältige Forschung und offene Zusammenarbeit zu dokumentieren und zu verstehen.**

# 🇬🇧 English

## Welcome

The **Swissphone Classic Archive** is an open-source community project dedicated to preserving historical Swissphone pagers.

Our goal is to document hardware, software, manuals, measurements and reverse engineering results so that this knowledge remains available for future generations.

Many historical Swissphone devices, programming tools and technical documents are becoming increasingly difficult to find. This project aims to preserve that knowledge before it is lost.

### Project Goals

- Document historical Swissphone devices
- Preserve software and manuals
- Reverse engineer hardware
- Publish reproducible measurements
- Develop open-source tools
- Encourage international collaboration

> **This project preserves technical history. Our goal is to document and understand historical Swissphone equipment through reproducible measurements, careful research and open collaboration.**
